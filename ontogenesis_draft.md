# SECTION 1: NOTATION GRAMMAR

## 1.1 Slot Structure

Each property fills one slot as it is ontogenetically determined. The reading order within a slot is **substrate → attitude**, matching OPS function notation naturally (Ni, Se, Te, Fi, etc.).

```
[_](_)__
  ↑   ↑ ↑
  |   | └── Substrate (T / F / S / N)
  |   └──── Attitude (i / e)
  └──────── Modality (m / f)

Stack position shown as [1]–[4] prefix once determined.
```

## 1.2 Undetermined Start State

```
_
```

Every derivation begins here. Each arrow represents one ontogenetic determination — one property resolving from undetermined potential into a fixed structural fact.

## 1.3 Step-by-Step Slot Filling

| Step | Notation | What resolved |
|------|----------|---------------|
| _ | `_` | Nothing determined |
| Attitude locks | `_i` or `_e` | Direction of libido |
| Substrate locks | `Ni` / `Te` / etc. | Mode of operation |
| Modality locks | `(m)Ni` / `(f)Te` / etc. | Holding quality under pressure |
| Stack position locks | `[1](m)Ni` | Ordonnance confirmed |
| Inferior resolves | `[4](f)Se` | Structural compensation complete |

## 1.4 Modality Shorthand (MM / FM / MF / FF)

The two-character modality code encodes the full stack's modality profile via two anchor points:

- **char_1** = modality of the sensory function (Se or Si, wherever it appears in the stack)
- **char_2** = modality of the extroverted decider (Te or Fe, wherever it appears)

Once both anchors are set, all four function modalities are mechanically determined by inversion rules.

| Code | Sensory Modality | Ext-Decider Modality |
|------|-----------------|----------------------|
| MM | Masculine | Masculine |
| MF | Masculine | Feminine |
| FM | Feminine | Masculine |
| FF | Feminine | Feminine |

## 1.5 Fixed Structural Pairings (Non-Negotiable)

These pairings are structural, not modality-based. They derive from Jung's complementarity principle.

**Observer axis:** Si ↔ Ne · Se ↔ Ni

**Decider axis:** Ti ↔ Fe · Fi ↔ Te

**[1]/[4] rule:** Same axis · Opposite attitude · Opposite modality

**[2]/[3] rule:** Opposite axis from [1]/[4] · Opposite attitude · Opposite modality

## 1.6 STACK_ANCHORS By Lead Type

|-----------|-----|-----|-----|-----|

| Introverted Decider (Ti / Fi) | ID | EO | IO | ED |

# SECTION 2: MODE 5.1 — ATTITUDE FOUNDATIONAL · SUBSTRATE COMPENSATES

## 2.1 Ontogenetic Steps

The Jungian canon. Attitude direction is the first ontological determination. Substrate differentiates within the established attitude. Modality assigns third. The [2]/[3] function choice and modality complete the stack.

```
_
```
**Step 1 — Attitude locks first.**
Adaptive pressure establishes the direction of libido before any mode of operation is determined.
```
_i  (or _e)
```
**Step 2 — Substrate determines.**
Within the established attitude, adaptive success selects which mode of engaging experience proves most reliable.
```
Ni  (example)
```
**Step 3 — Modality assigns.**
Holding quality under pressure resolves. The function is now fully specified as an Atomic Type.
```
(f)Ni
```
**Step 4 — Stack position locks.**
```
[1](f)Ni
```
**Step 5 — Inferior resolves by compensation.**
Substrate flips (N→S). Attitude alternates (i→e). Modality inverts (f→m).
```
[4](m)Se
```
**Step 6 — Middle functions branch.**
[2] and [3] are on the opposite axis, opposite attitudes, opposite modalities to each other.
```
[1](f)Ni
    ├── [2](f)Te  →  [3](m)Fi
    ├── [2](m)Te  →  [3](f)Fi
    ├── [2](f)Fe  →  [3](m)Ti
    └── [2](m)Fe  →  [3](f)Ti
         ↓
      [4](m)Se
```

**Explanatory priority:** The N/S flip between [1] and [4] is the *compensatory* determination. The i/e alternation is the *structural* consequence of attitude-foundational priority.

# SECTION 3: MODE 5.2 — ATTITUDE FOUNDATIONAL · MODALITY COMPENSATES

## 3.1 Ontogenetic Steps

Attitude direction is still the first determination. But modality — the holding quality under pressure — resolves second, before substrate is specified. Substrate differentiates last within the established attitude-modality configuration.

```
_
```
**Step 1 — Attitude locks first.**
```
_i  (or _e)
```
**Step 2 — Modality assigns second.**
Before substrate is known, the holding quality of the function resolves. The psyche is Introverted-Feminine (or Introverted-Masculine) before it is Intuitive or Sensing.
```
(f)i  (example)
```
**Step 3 — Substrate determines.**
The established attitude-modality configuration finds its best adaptive expression in a specific substrate.
```
(f)Ni
```
**Step 4 — Stack position locks.**
```
[1](f)Ni
```
**Step 5 — Inferior resolves by compensation.**
Modality inverts (f→m). Attitude alternates (i→e). Substrate flips (N→S).
```
[4](m)Se
```
**Step 6 — Middle functions branch.**
```
[1](f)Ni
    ├── [2](f)Te  →  [3](m)Fi
    ├── [2](m)Te  →  [3](f)Fi
    ├── [2](f)Fe  →  [3](m)Ti
    └── [2](m)Fe  →  [3](f)Ti
         ↓
      [4](m)Se
```

**Explanatory priority:** Modality is the *compensatory* determination. The [1]↔[4] modality opposition is explanatorily primary. The N/S substrate flip is a structural consequence, not the driver.

## 3.2 Mode 5.2 Tree Structure

Branching order: **Attitude → Modality at [1] → Substrate → [2] function → [2] modality**

```
_
├── _i
│   ├── (f)i → Ni / Si / Ti / Fi → [2] choice → (m)/(f) → stack
│   └── (m)i → Ni / Si / Ti / Fi → [2] choice → (m)/(f) → stack
└── _e
    ├── (f)e → Ne / Se / Te / Fe → [2] choice → (m)/(f) → stack
    └── (m)e → Ne / Se / Te / Fe → [2] choice → (m)/(f) → stack
```
